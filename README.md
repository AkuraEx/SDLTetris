### SDLTetris

## How To Play:
On a Windows machine just type out/tetris.exe in the main directory

## Controls:
* Left and Right to move
* Up/Z to rotate clockwise
* X to rotate counter clockwise
* Down to soft drop
* Space to hard drop
* C to hold a piece
